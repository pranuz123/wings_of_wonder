Wings of Wonder — color-guide photos
=====================================

Drop one photo per bird in THIS folder, named exactly like the bird's slug,
all lowercase, with a .jpg extension. The gallery (index.html) will show them
automatically. Until a file exists, that bird shows a friendly placeholder.

Required file names:
  photos/owl.jpg
  photos/toucan.jpg
  photos/robin.jpg
  photos/cardinal.jpg
  photos/blue-jay.jpg
  photos/peacock.jpg
  photos/flamingo.jpg
  photos/hummingbird.jpg
  photos/kingfisher.jpg
  photos/puffin.jpg
  photos/hoopoe.jpg
  photos/penguin.jpg
  photos/pelican.jpg
  photos/swan.jpg
  photos/hornbill.jpg

Tips
----
- Use photos you are allowed to use. Great free, properly-licensed sources:
  Wikimedia Commons, Unsplash, Pixabay, Pexels. Prefer public-domain (CC0) or
  Creative Commons images, and credit the photographer if the license asks.
- Landscape photos around 1200x900 px (4:3) look best. JPG, under ~500 KB each.
- .jpg only (lowercase). If you must use .png, rename the reference in index.html.
